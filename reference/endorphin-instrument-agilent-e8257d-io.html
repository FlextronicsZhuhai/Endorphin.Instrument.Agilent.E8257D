Endorphin.Instrument.Agilent.E8257D

IO

Namespace: Endorphin.Instrument.Agilent.E8257D

Common functions to set/query values of a VISA Agilent instrument. Includes functions to access values such as numbers, frequencies etc., which are common to different subsystems.
